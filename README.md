# 🚀 راهنمای اجرا

## پیش‌نیازها

* Java JDK (نسخه 17 یا بالاتر)
* IntelliJ IDEA
* کتابخانه SQLite JDBC

## نحوه اجرا

1. پروژه را در نرم‌افزار **IntelliJ IDEA** باز کنید.
2. در صورت نیاز، کتابخانه **SQLite JDBC** را به پروژه اضافه کنید.
3. کلاس `Main` را اجرا کنید.
4. در اولین اجرای برنامه، فایل پایگاه داده (`game.db`) به صورت خودکار ایجاد خواهد شد.

# 🗄️ پایگاه داده

## نوع پایگاه داده

**SQLite**

## مسیر فایل پایگاه داده

```
game.db
```

## جدول‌های پایگاه داده

### جدول `users`

این جدول اطلاعات کاربران بازی را ذخیره می‌کند.

| نام ستون          | توضیح                           |
| ----------------- | ------------------------------- |
| `username`        | نام کاربری (کلید اصلی)          |
| `password`        | رمز عبور                        |
| `highest_score`   | بیشترین امتیاز کسب‌شده          |
| `highest_level`   | بیشترین مرحله طی‌شده            |
| `coins`           | تعداد سکه‌های بازیکن            |
| `unlocked_planes` | لیست هواپیماهای خریداری‌شده     |
| `equipped_plane`  | هواپیمای انتخاب‌شده توسط بازیکن |

### توضیحات

در این پروژه از پایگاه داده **SQLite** برای ذخیره اطلاعات کاربران استفاده شده است. پایگاه داده در اولین اجرای برنامه به صورت خودکار ایجاد می‌شود و اطلاعات مربوط به کاربران در آن ذخیره می‌شود.

موارد ذخیره‌شده در پایگاه داده عبارت‌اند از:

* ثبت‌نام کاربران
* ورود کاربران
* بیشترین امتیاز کسب‌شده
* بیشترین مرحله طی‌شده
* تعداد سکه‌های بازیکن
* هواپیماهای خریداری‌شده
* هواپیمای انتخاب‌شده
* جدول ۱۰ بازیکن برتر
