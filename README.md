# پروژه Chicken Invaders

## مشخصات دانشجو
- نام و نام خانوادگی: پانیذ بلوری
- شماره دانشجویی: 40413407

## راهنمای اجرا
1. پروژه را در IntelliJ IDEA باز کنید.
2. در صورت نیاز، وابستگی SQLite JDBC را به پروژه اضافه کنید.
3. کلاس `Main` را اجرا کنید.
4. در اولین اجرا، فایل پایگاه داده (`game.db`) به صورت خودکار ایجاد می‌شود.

## کنترل‌های بازی
- حرکت به بالا: `W` یا `↑`
- حرکت به پایین: `S` یا `↓`
- حرکت به چپ: `A` یا `←`
- حرکت به راست: `D` یا `→`
- شلیک: `Space`
- توقف یا ادامه بازی: `P`
- باز کردن منوی تنظیمات: `M`

## پایگاه داده

**نوع پایگاه داده:** SQLite

**مسیر فایل پایگاه داده:** `game.db`


**جدول‌ها:**

### users
این جدول اطلاعات کاربران را ذخیره می‌کند و شامل موارد زیر است:
- نام کاربری
- رمز عبور
- بیشترین امتیاز
- بیشترین مرحله طی‌شده
- تعداد سکه‌ها
- لیست هواپیماهای خریداری‌شده
- هواپیمای انتخاب‌شده

**توضیحات:**

در این پروژه از پایگاه داده SQLite برای ذخیره اطلاعات کاربران استفاده شده است. پایگاه داده در اولین اجرای برنامه به صورت خودکار ایجاد می‌شود. اطلاعاتی مانند ثبت‌نام و ورود کاربران، بیشترین امتیاز، بیشترین مرحله، تعداد سکه‌ها، هواپیماهای خریداری‌شده، هواپیمای انتخاب‌شده و جدول ۱۰ بازیکن برتر در این پایگاه داده ذخیره می‌شوند. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1}
